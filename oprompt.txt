IMPORTANTE:
Não gere código inicialmente. Antes de qualquer implementação, crie um planejamento completo da aplicação, definindo arquitetura visual, experiência do usuário, componentes e fluxo de navegação.

## Objetivo do Projeto

CodeVerse é uma aplicação desktop desenvolvida com Electron, HTML, CSS e JavaScript que transforma os repositórios GitHub de um desenvolvedor em um universo espacial interativo.

Cada repositório é representado por um planeta.

O sistema deve consumir a API do GitHub para buscar automaticamente os repositórios de um usuário e exibi-los visualmente em uma galáxia personalizada.
